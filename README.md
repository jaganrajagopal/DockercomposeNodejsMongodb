# DockercomposeNodejsMongodb#
To create the docker compose with multiple container with Nodejs as frontend and Backend as Mongodb 


**Step 1: Please clone the Project**

**Step 2: Please run this commands on local machine on this current directory**

mkdir node-mongo-restapi && cd node-mongo-restapi
npm init -y
